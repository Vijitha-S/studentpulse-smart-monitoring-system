# 🎓 StudentPulse – Smart Student Monitoring System

## 📌 Overview

StudentPulse is a full-stack web-based application designed to manage and monitor student academic data efficiently. It provides a centralized platform for students, staff, and administrators to interact with academic information such as marks, attendance, and fee status in real time.

---

## 🎯 Features

### 👨‍💼 Admin Module

* Add / Manage Departments
* Add / Manage Years
* Add / Manage Subjects (Dynamic)
* Add / Manage Students
* Delete and update records

### 👨‍🏫 Staff Module

* View student details
* Enter marks dynamically based on department & year
* Update attendance and fees
* View analytics dashboard

### 🎓 Student Module

* View subject-wise marks
* View attendance percentage
* View fee status
* View rank and performance
* Performance chart (Chart.js)
* Warning alerts (low marks, attendance, fees)

## 🚀 Installation & Setup

### 1. Install XAMPP

Download and install XAMPP.

### 2. Move Project

Place project folder inside:

```
C:\xampp\htdocs\
```

### 3. Start Server

Start:

* Apache
* MySQL

### 4. Import Database

* Open phpMyAdmin
* Create database: `college_system`
* Import SQL file

### 5. Run Project

Open browser:

```
http://localhost:8080/studentpulse/
```
